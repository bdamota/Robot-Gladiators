# ROBOT GLADIATORS 

## PURPOSE:
  A game that pits the user robot against the computers' challengers. 
  
## MADE WITH:
 * HTML5
 * JavaScript  
  
## WEBSITE FUNCTIONALITY:
Uses a series of window prompts to move player through battles with a new opponent in each round.
